## Installation

1. Clone this repository:
   ```bash
    git clone https://github.com/valentinfrlch/camera-control
    ```
1. Install gphoto2
    ```bash
    brew install gphoto2
    ```
2. Install Python dependencies
    ```bash
    cd camera-control
    pip install -r requirements.txt
    ```
3. Connect your camera via USB
4. Run webserver
    ```bash
    python webserver.py
    ```
